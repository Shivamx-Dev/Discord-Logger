# Discord Logger Assets

This directory contains the frontend assets for the Discord Logger plugin.

## Structure

```
assets/
├── css/
│   └── admin-style.css    # Admin panel styling
├── js/
│   └── admin-script.js    # Admin panel JavaScript functionality
└── README.md              # This file
```

## CSS Features

- Responsive tabbed interface
- Modern card-based layout
- Loading animations
- Status indicators
- Mobile-friendly design

## JavaScript Features

- Tab navigation
- AJAX form handling
- Real-time webhook validation
- Settings import/export
- Auto-refresh logs
- Keyboard shortcuts
- Notification system

## Browser Support

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Dependencies

- jQuery (included with WordPress)
- WordPress admin styles
